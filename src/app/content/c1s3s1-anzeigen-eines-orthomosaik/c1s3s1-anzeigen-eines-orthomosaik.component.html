Bei einem Orthomosaik handelt es sich um eine Abbildung, welche aus mehreren Fotos (Mosaik) erzeugt wurde. Charakteristisch für ein Orthomosaik ist, dass alle perspektivischen Verzerrungen durch eine Orthogonalprojektion entfernt werden. Dies bedeutet, dass jederzeit eine Draufsicht auf das Orthomosaik dargestellt wird. Zur Veranschaulichung wurden zwei Gebiete mit unterschiedlicher Auflösung, Ausdehnung und Quelle genutzt, um auf Besonderheiten, welche diese Eigenschaften hervorrufen, eingehen zu können. Dies ermöglicht einen Vergleich zwischen einem Low-Cost Ansatz mittels Drohne und einem High-Cost Ansatz mittels Gyrocopter.

Das erste Gebiet ist ein Industriegebiet mit dem Namen Lindenrain, welches aktuell von der Stadt Calw erschlossen wird. Die Daten wurden durch eine Drohnenbefliegung in 120 Meter Höhe erfasst. Genutzt wurde hierfür eine Drohne des Herstellers DJI mit der Modellbezeichnung Matrice 300 RTK und als Sensor eine Kamera von DJI welche speziell für die Luftbildvermessung geeignet ist, nämlich das Modell Zenmuse P1 mit 44,72 Megapixel Auflösung. Dieser Datensatz bedeckt eine Fläche von 26,37 Hektar bei einer Bodenauflösung von 2 cm pro Pixel. Der Datensatz besteht aus 4 Kanälen (RGBA) mit dem Datentype Byte (Werte zwischen 0 und 255) und einer Datenmenge von 6,9 Gigabyte. Das Koordinatensystem des Datensatzes ist "WGS 84 / Web Mercator" (EPSG:3857).

Das zweite Gebiet mit der Bezeichnung Strenzfeld wurde mittels Gyrocopter der Hochschule Anhalt Institut für Geoinformation und Vermessung in einer Flughöhe von 545 Meter erfasst. Als Sensor wurde eine iXM-100 mit 102,06 Megapixel pro Bild genutzt. Der Datensatz umfasst eine Fläche von 440,18 Hektar bei einer Bodenauflösung von 5 cm pro Pixel. Weiterhin besteht er aus 3 Kanälen (RGB) mit dem Datentype uint16 (Werte zwischen 0 und 65535) und einer Datenmenge von 16,0 Gigabyte. Das Koordinatensystem des Datensatzes ist "WGS 84 / UTM zone 32N" (EPSG:32632).
