Um eine Nutzung zu ermöglichen, müssen die Eingangsdaten für einen Anwender aufgearbeitet werden. Dies geschieht in mehreren Arbeitsschritten. Jeder dieser Schritte wird nachfolgend in einem separaten Kapitel beschrieben.

Die Daten müssen zuerst für die unterschiedlichen Standards vorbereitet werden. Dazu zählt die Vorbereitung der Datensätze für die jeweiligen Tools, sowie die Erstellung der Dateien für die Standards. Die Vorgehensweise wird in Kapitel 3.1 erläutert.

Nachdem die Dateien erstellt wurden, müssen diese über einen Webserver bereitgestellt werden, was innerhalb eines lokalen Netzwerkes, einem Intranet oder dem Internet erfolgen kann. Innerhalb dieser Arbeit werden Amazon Web Services (AWS) verwendet. Hierauf wird in Kapitel 3.2 eingegangen.

Im Anschluss an die Bereitstellung der Daten durch einen Webserver, wird in Kapitel 3.3 auf die Nutzung in Form der Implementierung der Standards in eine WebMap eingegangen. Ziel hierbei ist es, die Szenarien aus Kapitel 1.3 in einem realen Anwendungsfall darzustellen.
