Die in Kapitel 3.1.1 erzeugten TMS liegen nun lokal auf der Festplatte. Die Auslieferung über ein Netzwerk wäre jedoch nicht möglich und somit der Nutzerkreis auf einen Arbeitsplatz beschränkt. Ziel ist es, die Daten über das Internet den Nutzern zur Verfügung zu stellen. Hierzu werden die Daten auf einem WebServer, welcher HTTP-GET-Requests akzeptiert, platziert.

Das Ordnerverzeichnis der einzelnen TMS wird über den folgenden AWS CLI- Befehl auf das hierfür vorgesehene AWS S3- Bucket geladen.
